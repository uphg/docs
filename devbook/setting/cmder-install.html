# 安装 Cmder

注意事项

  • 如果安装了浏览器FQ插件（如 Proxy SwitchyOmega），下载压缩包时请在一个无痕的窗口打开，并开启全局代理，下载速度会更快。
  • 如果安装电脑中前已经有 Git Bash 直接下载安装简化版即可，此教程不包括简化版安装教程

下载

  • 官网链接：https://cmder.net/
  • 下载后尽量解压到一个非中文无空格路径下的目录

配置

  1. 首先解压文件 (最好放在一个没有空格的目录)，解压后打开

    • 打开设置，选择导出，如图：

设置分屏

在设置中选择 按键 & 宏 一栏在其中的搜索框中搜索 分屏 然后设置 复制分屏至底部和复制分屏至右侧的快捷键，如图：

设置之后就可以使用快捷键分屏了

# 添加主题

在 setting 的 特征 - 颜色 中，随便修改一个主题的名字并保存，它就会生成一个新的自定义主题，在 ConEmu.xml 中搜索并替换该主题的配置为你想要的配置即可。
