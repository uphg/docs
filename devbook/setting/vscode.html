# 自动保存/格式化

Ctrl + Shift + P 搜索 Settings 选择：打开设置(ui)，在设置中搜索以下内容进行配置

  1. 设置自动保存：auto save
  2. 保存时自动格式化：format on save

# 配置根据 ESLint 自动格式化

Ctrl + Shift + P 搜索 Settings 选择：打开设置(json)，在设置中添加以下配置

{
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "editor.formatOnSave": false // 关闭编辑器自带保存格式化功能
}

# 配置 git 及命令行

使用 Ctrl + Shift + P 搜索 Setting，选择：打开设置(json)，添加以下三行配置

{
  "git.enabled": true, // 表示使用自己配置的 git
  "git.path": "C:\\xxx\\cmder\\vendor\\git-for-windows\\cmd\\git.exe", // 配置 git 的路径
  "terminal.integrated.shell.windows": "C:\\xxx\\cmder\\vendor\\git-for-windows\\bin\\bash.exe", // 配置 bash 的路径
}
