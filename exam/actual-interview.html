# 柯里化

下面的过程就是柯里化（一个函数返回一个函数）

const add = (x, y) => x + 2 * y
add(1, 2) // 5

// 这里的 fn 叫「偏函数」
const fn = (y) => add(1, y)
fn(2)     // 5

推荐一个函数库：Underscore.js (opens new window)，查找它的 partial 方法，该方法可以将一个函数柯里化。

# 高阶函数

只要满足以下条件之一，就是高阶函数

  • 接受一个或多个函数作为输入
  • 输出一个函数

JavaScript 中典型的内置高阶函数：Function.prototype.bind()

# LazyMan 面试题

实现 LazyMan，能按以下方式调用

  1. 打印出一个 log
LazyMan("Hank")

你好，我是 Hank
  1. 可以链式调用
LazyMan("Hank").eat("lunch").eat("supper")

你好，我是 Hank
吃午餐
吃晚餐
  1. 可以指定时间后执行后续的内容
LazyMan("Hank").sleep(10).eat("lunch")

你好，我是 Hank
（沉默十秒）
我醒了，我刚睡了 10 秒
吃午餐
  1. 可以根据后面链式调用的方法控制代码最开始执行的时间
LazyMan("Hank").sleepFirst(5).eat("supper")

（沉默五秒）
我醒了，我刚睡了 5 秒
你好，我是 Hank
吃晚餐

答案

实现前两个功能

function LazyMan(name) {
  console.log(`你好！我是 ${name}`)
  const api = {
    eat(type) {
      const hash = {
        'lunch': '吃午餐',
        'supper': '吃晚餐'
      }
      console.log(hash[type])
      return api
    }
  }
  return api
}

LazyMan('小明').eat('lunch').eat('supper')

你好！我是小明
吃午餐
吃晚餐

实现第3题

function LazyMan(name) {
  console.log(`你好！我是 ${name}`)
  const api = {
    sleep(n) {
      const time = Date.now()
      while(Date.now() - time < n*1000){}
      console.log(`我醒了，我刚睡了${n}秒`)
      return api
    },
    eat(type) {
      const hash = {
        'lunch': '吃午餐',
        'supper': '吃晚餐'
      }
      console.log(hash[type])
      return api
    }
  }
  return api
}

LazyMan('小明').sleep(10).eat('supper')

实现第4题

function LazyMan(name) {
  const queue = []
  const task = () => { console.log(`你好！我是 ${name}`); next() }
  queue.push(task)
  const next = () => {
    const first = queue.shift()
    first && first() // first?.()
  }
  const api = {
    _queue: queue,
    sleep(n) {
      const task = () => {
        setTimeout(() => {
          console.log(`我醒了，我刚睡了${n}秒`)
          next() 
        }, n*1000)
      }
      queue.push(task)
      return api
    },
    eat(type) {
      const task = () => {
        const hash = {
          'lunch': '吃午餐',
          'supper': '吃晚餐'
        }
        hash[type] && console.log(hash[type])
        next()
      }
      queue.push(task)
      return api
    },
    sleepFirst(n) {
      const task = () => {
        setTimeout(() => {
          console.log(`我醒了，我刚睡了${n}秒`)
          next() 
        }, n*1000)
      }
      queue.unshift(task)
      return api
    }
  }
  setTimeout(() => { next() })
  return api
}

LazyMan("小明").sleepFirst(5).eat("supper")

其中的 _queue 只是为了知道当前任务队列的任务，可不写。

# 使用函数实现循环

实现一个数组，数组内值按照顺序每次都加 3，并且可以指定数组长度。

  1. 范围和初始值都使用参数控制。
  2. 每次+3。

代码

function addArray(number, max) {
  let array = []
  function fn(number, max) {
    if (max < array.length + 1) {
      return
    }
    number += 3
    array.push(number)
    return fn(number, max)
  }
  fn(number, max)
  return array
}
console.log(addArray(0, 10))
